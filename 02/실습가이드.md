# 02. RR / Max-CQI / PF 비교

## 목표
동일 UE·동일 CQI 시퀀스에서 Round Robin, Max-CQI, PF를 돌리고  
총 throughput과 Jain fairness를 비교한다.

## Jain index
```
J = (sum r_i)^2 / (N * sum r_i^2)
```
1에 가까울수록 공평, 1/N에 가까울수록 불평등.

## 실습 과제
1. `ALGO`를 `ALGO_RR` / `ALGO_MAX_CQI` / `ALGO_PF`로 바꿔가며 실행
2. `summary` 표의 total_bytes, Jain 값을 메모
3. Max-CQI에서 어느 UE가 독점하는지 관찰
4. (선택) `seed`를 바꾸면 fairness가 어떻게 변하는지 확인
