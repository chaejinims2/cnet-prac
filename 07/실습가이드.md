# 07. Weighted PF / M-LWDF 분리 비교

## 목표
04의 metric을 쪼개 **weight만 / delay만 / 둘 다** 세 경우를 같은 seed로 비교한다.

## metric
```
weight_only : w * PF
delay_only  : (hol_delay+1) * PF
both        : w * (hol_delay+1) * PF
```

## 실습 과제
1. `ue1_picks`(Non-GBR)가 mode별로 어떻게 달라지는지 표로 확인
2. 04 delay ON/OFF와 숫자가 어떻게 대응하는지 연결
